# aurora-delivery-platform

Aurora Delivery Platform is a cloud-native microservices application designed to demonstrate a production-grade CI/CD pipeline using modern DevOps practices.

Key Features
- Microservices-based architecture
- Jenkins-driven CI/CD pipelines
- Kubernetes deployments using Helm
- Automated testing, security scanning, and monitoring
- Production-ready Git workflow
